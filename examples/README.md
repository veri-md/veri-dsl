# Veri DSL Examples

Example `.veri.md` files written in Veri DSL (F*Veri DSL).  
These replace traditional F* `veri` code blocks with our DSL syntax.

## Examples

| File | Demonstrates |
|---|---|
| `sorted_list.veri.md` | Records, refined types, predicates, simple contracts |
| `circular_buffer.veri.md` | Constants, invariants, `option` return types, match expressions |
| `lru_cache.veri.md` | Quantifiers (`FORALL`/`EXISTS`), `nth_opt`, timestamp reasoning |
| `fairinf_core.veri.md` | Custom C type mapping (`int32`, `float64`, `string(128)`), `IN OUT` params, `CONSTRAINT` blocks, array contracts |
| `circular_buffer_heap.veri.md` | (TO COME) ST monad, heap refs, `live`, `modifies`, `Buffer.get` |
